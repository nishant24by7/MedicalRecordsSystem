## 📌 Project Description
The Medical Records System is a decentralized application built on the Stellar Soroban smart contract platform. It enables secure storage and retrieval of patient medical records on-chain, ensuring transparency, integrity, and ownership control.

## ⚙️ What it does
This smart contract allows:
- Patients or healthcare providers to store medical records.
- Retrieval of medical records using a unique ID.
- Verification of record ownership.

Each medical record is linked to a patient’s address, ensuring that ownership is clearly defined and immutable.

## 🚀 Features
- 🔐 **Decentralized Storage** – Records are stored on-chain.
- 👤 **Ownership Tracking** – Each record is linked to a patient address.
- 🔍 **Easy Retrieval** – Fetch records using unique IDs.
- 📈 **Auto Increment IDs** – Each record gets a unique identifier.
- ⚡ **Lightweight & Simple** – Basic implementation for easy extension.

## 🧠 Future Improvements
- Encrypt medical data for privacy
- Role-based access (doctors, hospitals)
- Record updates & history tracking
- IPFS integration for large files
